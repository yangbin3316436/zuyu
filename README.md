# 天上人间洗浴中心 Next.js 官网

这是由原静态 HTML 页面迁移而来的 Next.js App Router 项目。

## 运行

```bash
npm install
npm run dev
```

然后打开：

```text
http://localhost:3000
```

## 页面

- 首页：`/`
- 魅影迎宾礼遇：`/projects/glam-hostess`
- 午夜香薰SPA：`/projects/aroma-spa`
- 玫瑰足浴护理：`/projects/rose-footcare`
- VIP茶歇包间：`/projects/vip-lounge`

## 构建检查

```bash
npm run build
```
